LeetCode 1137. N-th Tribonacci Number

Link

Description

The Tribonacci sequence Tn is defined as follows:
T0 = 0, T1 = 1, T2 = 1, and Tn+3 = Tn + Tn+1 + Tn+2 for n >= 0.
Given n, return the value of Tn.

Example

Example 1:

Input: n = 4
Output: 4
Explanation:
T_3 = 0 + 1 + 1 = 2
T_4 = 1 + 1 + 2 = 4
Example 2:

Input: n = 25
Output: 1389537

Tags:
Dynamic Programming(DP)

Solve 1

此題跟費波那契數列解題方式基本一樣，使用Recursive方式解題。

不過這段程式碼在測資是不過的，時間複雜度超過了。

  • Time complexity: $O(3^n)$ 每個項下面都會分成3個項目，以Tree方式看其每一個子運算都會以指數量級增長
  • Space complexity: $O(n)$ 需要n層來儲存調用Function的狀態.

Solve 2

先初始化基本條件，藉由陣列索引的方式來做DP。

  • Time complexity: $O(n)$ 一個迴圈就搞定了。
  • Space complexity: $O(n)$ 需要長度為n+1的陣列來儲存運算好的數值。

結論

在實作上都不要使用Recursive避免時間複雜度爆炸影響效率。
