define-and-read-in-c

Posted on 2023-12-16 In C

Introduction

解讀C語言的宣告式，平常看起來不太重要，但面對一些特殊的宣告式我們仍要有一個辦法去解讀他。

Right-left Rule

Right-left Rule指得是解讀C語言宣告式的一個法則。

符號

以下有3個基本的運算子，

  • *讀為 “指標指向的是” “pointer to” 只會在 (辨識字) 左邊出現
  • []讀為 “陣列元素為” “array of” 只會在 (辨識字) 右邊出現
  • ()讀為 “函數回傳值” “function returning” 只會在 (辨識字) 右邊出現

辨識子就是函數名稱或變數名稱喔

步驟

  1. 找到辨識字，解讀為 xxx是 或者 xxx is
  2. 往辨識字的右邊看，如果看到
    • [] 解讀為 xxx是陣列，其元素為 xxx is the array of …
    • () 解讀為 xxx是函數, 其回傳值為 xxx is the function returning …
      直到遇到 單獨的 )
    • 如果遇到(則代表中間有其他東西要解讀
  3. 接著往辨識字的左邊看
    • * 解讀為 指向的是， pointer to
    • datatype 就照念就好。

char-in-c

Posted on 2023-12-16 In C

Introduction

在C語言中，如果要宣告一個字串是需要透過矩陣進行宣告的，不能像C++那樣去宣告string

  • 宣告字串陣列，不給定長度情況下, 結尾可能出現亂碼，由於沒有結束字符。
  • 宣告字串陣列, 不給定長度形況下, 這樣才是安全的宣告方式
  • 宣告字串陣列, 使用兩個引號, 編譯器會自動加上’\0’, 非常安全以及好用
  • 宣告字串陣列,給定長度以及初始化的值
